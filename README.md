# discord_sample
ディスコードで遊んでみる

1. 仮想環境を作成                                                                                                                                                         
                                                                                                                                                                        
python3 -m venv venv                                

2. 仮想環境を有効化

source venv/bin/activate

仮想環境が有効になると、プロンプトの先頭に (venv) と表示されますわ。

3. 依存パッケージをインストール

pip3 install -r requirements.txt

4. 作業が終わったら仮想環境を無効化

deactivate
